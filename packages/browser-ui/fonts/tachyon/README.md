# Tachyon Webview Font

This directory vendors the Tachyon mono UI font used by webviews.

- Family: JetBrains Mono
- Version: 2.304
- Source: https://github.com/JetBrains/JetBrainsMono/releases/tag/v2.304
- License: SIL Open Font License 1.1 (`OFL.txt`)
- Bundled weights: Regular 400, Medium 500, SemiBold 600, Bold 700

Build output copies these files to `dist/webview/fonts/tachyon` so webviews can load them through VS Code's local resource URI and CSP.
